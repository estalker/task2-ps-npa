По дате подписания НПА: Официальный реквизит правового акта. Для данного поля существуют две опции: выбрать точную дату, выбрать диапазон дат. Ввод даты можно осуществлять: с клавиатуры в соответствующие поля в формате дд.мм.гггг; посредством элемента управления «календарь», по кнопке, расположенной справа от текстового поля дд.мм.гггг.
По дате опубликования НПА: Дата официального опубликования правового акта на официальном интернет-портале правовой информации. Для данного поля существуют две опции: выбрать точную дату, выбрать диапазон дат. Ввод даты можно осуществлять: с клавиатуры в соответствующие поля в формате дд.мм.гггг. посредством элемента управления «календарь», по кнопке, расположенной справа от текстового поля дд.мм.гггг.
